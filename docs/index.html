CURRENT PROJECTS

Habit Tracking App

Built using C# ASP.NET
My Habbits is a CRUD application that allows users to create, read, update, and delete tasks.

Each task is saved onto a Microsoft Azure hosted MSSQL Database. Completed tasks will be saved to a completed_history table, with the use of Microsoft SQL Server 2016. An SQL query runs at the end of the day. This query saves all completed tasks to the completed_history table and resets data from the tasks table.

Tools Used : C#, ASP.NET MVC, EntityFramework, MSSQL, SQL Server 2016, Razor, Bootstrap 3, Javascript, JQuery, AJAX, Git, Microsoft Azure.
